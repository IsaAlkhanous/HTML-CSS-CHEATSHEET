CSS Selectors

Selectors are used to select the HTML elements you want to style.

  • Universal Selector: * { ... } (Selects all elements)
  • Element Selector: p { ... } (Selects all <p> elements)
  • Class Selector: .classname { ... } (Selects elements with a specific class)
  • ID Selector: #idname { ... } (Selects an element with a specific ID)
  • Attribute Selector: a[href] { ... } (Selects <a> elements with an href attribute)
  • Descendant Selector: ul li { ... } (Selects <li> elements inside <ul>)
  • Child Selector: div > p { ... } (Selects <p> elements directly under a <div>)
  • Adjacent Sibling Selector: h2 + p { ... } (Selects <p> elements immediately after <h2>)

CSS Box Model

The box model consists of margins, borders, padding, and the actual content.

div { width: 300px; padding: 20px; border: 5px solid gray; margin: 15px; }

Typography

Styling text content using CSS.

  • font-family: Sets the font type.
  • font-size: Sets the size of the font.
  • font-weight: Sets the thickness of the text.
  • line-height: Sets the space between lines.
  • text-align: Aligns text horizontally.
  • text-decoration: Adds decoration like underline.
  • text-transform: Transforms text casing.
p { font-family: 'Arial', sans-serif; font-size: 16px; line-height: 1.5; text-align: justify; }

Colors & Units

Colors

  • Named Colors: color: red;
  • Hexadecimal: color: #ff0000;
  • RGB: color: rgb(255, 0, 0);
  • RGBA: color: rgba(255, 0, 0, 0.5);
  • HSL: color: hsl(0, 100%, 50%);

Units

  • Absolute Units: px, pt, cm, mm, in
  • Relative Units: em, rem, %, vw, vh

Transitions & Animations

Transitions

Smoothly change property values over time.

Hover over me!
/* Transition Example */ .transition-example { background-color: lightblue; padding: 20px; transition: background-color 0.5s; /* Transition effect */ } .transition-example:hover { background-color: lightsalmon; /* New color on hover */ }

Animations

Create complex animations using keyframes.

Watch me move!
/* Animation Example */ @keyframes move { from { transform: translateX(0); } to { transform: translateX(100px); } } .animation-example { animation: move 2s infinite alternate; }

Responsive Design

Ensure your website looks good on all devices using media queries.

/* Media Query Example */ @media (max-width: 600px) { body { background-color: lightgray; } }
